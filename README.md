# PDFQuery

## Overview
**PDFQuery** is a user-friendly application designed to allow users to upload PDF documents and interact with their content through a chatbot powered by OpenAI's LLM.

---

## Features

### Core Features
#### **Chat with Your PDFs**
- Upload PDFs and ask questions based on their content.
- Get detailed, accurate answers directly from the text.

#### **Multi-Document Upload**
- Upload and process multiple documents at once.
- Combine insights across different files seamlessly.

#### **Advanced Prompt Engineering**
- Answer specialized questions (e.g., financial queries like *"What was the revenue in 2023?"*).
- Generate document summaries or extract key facts.
- Improve text formatting for tables and structured data.

#### **Customizable LLM Parameters**
- Adjust parameters like `temperature` and `max_tokens` to control the creativity and verbosity of the responses.

#### **Chat History Management**
- View and manage the history of queries and responses.
- Maintain context for follow-up questions during a session.

### Enhanced Scalability
- **Modular Pipelines**: For document processing (e.g., text extraction, cleaning, and splitting).
- **Containerized Application**: Easily deploy and scale with Docker.

---

## Installation

### Prerequisites
- Python 3.8 or above
- Docker (optional for containerized deployment)
- OpenAI API key (for LLM integration)
